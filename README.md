## Descripción

Sistema de gestión de clientes, servicios y reservas para la empresa Software FJ desarrollado en Python utilizando Programación Orientada a Objetos.

## Características

* Gestión de clientes
* Gestión de servicios
* Gestión de reservas
* Manejo de excepciones personalizadas
* Registro de eventos mediante logs
* Implementación de abstracción, herencia, polimorfismo y encapsulación

## Requisitos

Python 3.10 o superior

## Ejecución

```bash
python src/main.py
```

## Principios POO implementados

* Abstracción
* Herencia
* Polimorfismo
* Encapsulación

## Manejo de excepciones

* try/except
* try/except/else
* try/finally
* Encadenamiento de excepciones
